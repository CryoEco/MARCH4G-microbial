**File descriptions**

*16S-tree.R*: Code to generate a phylogenetic tree for the whole dataset, which was used for the phylogenetic null modeling (iCAMP).

*Script_16S_betadiv.R*: Code to run the Principal Coordinates Analysis (PCoA), PERMANOVA, dispersion test, and to generate Figure 2.

*Script_16S_data_prep_phyloseq_object.R*: Code to generate the phyloseq object, do the rarefaction, and prepare the phyloseq object for the iCAMP analysis.

*Script_ANCOM_BC.R*: Code to run the differential abundance analysis and generate Figure 5.

*Script_DADA2_pipeline.R*: Code to run the DADA2 pipeline and create the ASV count table and the taxonomy count table.

*Script_dbRDA.R*: Code to run the distance-based redundancy analysis and generate Figure 3.

*Script_icamp.R*: Script to run the phylogenetic bin-based null model analysis iCAMP.

*Script_seq_prep_predada2.ipynb*: Code to check sequence quality and cut primers.

*batch_fastp_avgQ30.sh*: Code to filter out reads that have an average phred score <30.

*Script_CH4trophs_CH4gens_in_CH4ranges_fig4.R*: Script to generate Figure 4.

*Script_icamp_figure_6.R*: Script to generate Figure 6.

*Script_Methane_ranges_Supp_fig_1.R*: Script to generate Supplementary Figure 1.
